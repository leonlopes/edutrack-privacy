Política de Privacidade — Double L Classes

Este é um documento em revisão contínua. Não constitui aconselhamento jurídico — está em processo de validação formal, especialmente por envolver dados de menores de idade.

1. Quem somos

O Double L Classes é uma escola de idiomas. Este documento explica como coletamos, usamos e protegemos os dados pessoais de alunos e responsáveis, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

2. Quais dados coletamos

  • Dados de identificação: nome, sobrenome, idade
  • Dados de contato: telefone, e-mail (do aluno e, quando aplicável, do responsável)
  • Dados acadêmicos: presença, conteúdo das aulas, notas, boletins, homework
  • Dados financeiros: valor da mensalidade, forma de pagamento, histórico de pagamentos
  • Dados fiscais (quando aplicável): CPF/CNPJ, endereço, para emissão de Nota Fiscal

3. Para que usamos esses dados

  • Organização de aulas, presença e conteúdo pedagógico
  • Cálculo de notas e envio de boletins
  • Cobrança e controle financeiro, incluindo lembretes de pagamento
  • Comunicação sobre aulas, homework e assuntos administrativos
  • Emissão de Nota Fiscal, quando solicitado

4. Com quem compartilhamos

Não compartilhamos dados com terceiros para fins comerciais. Os dados são acessados apenas pelos professores responsáveis (Leo e Letícia) através do sistema interno (EduTrack), hospedado no Google Firebase.

Mensagens de aviso (mensalidade, homework, boletim) são enviadas via WhatsApp ou e-mail diretamente pelo professor ao aluno/responsável — nenhuma dessas mensagens passa por serviços de terceiros além do WhatsApp/e-mail do próprio destinatário.

5. Dados de menores de idade

Quando o aluno é menor de idade, priorizamos o contato com o responsável legal para comunicações administrativas e financeiras. A coleta de dados de menores é feita no contexto da prestação do serviço educacional contratado pelo responsável.

6. Segurança

Os dados são armazenados com controle de acesso restrito, apenas para os professores autorizados.

7. Seus direitos

Você pode, a qualquer momento, solicitar:

  • Confirmação de quais dados temos sobre você/seu(sua) filho(a)
  • Correção de dados incorretos
  • Exclusão dos dados, ao encerrar o vínculo com a escola

Solicitações podem ser feitas diretamente com Leo ou Letícia.

8. Retenção de dados

Mantemos os dados enquanto durar o vínculo do aluno com a escola, e por um período razoável após o término, para fins de histórico acadêmico e obrigações fiscais.

9. Contato
